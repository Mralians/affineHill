Encryption and Decryption of File Using Affine Hill Ciphering

## Compiling

```plaintext
git clone https://github.com/Mralians/affineHill.git
cd affineHill
make
```

## Usage

```plaintext
./keyGen or ./keyGen -d (for more details)
./affinehill [OPTION]... keyfile.txt SOURCE DECT
               -e  for encryption
               -d  for decryption
```
